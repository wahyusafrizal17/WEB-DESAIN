Membuat Laporan PDF Dengan PHP dan MySQLi

Pada kesempatan baik ini kami akan memberikan sebuah tutorial tentang bagaimana cara membuat laporan PDF dengan php dan mysql, membuat sebuah report atau laporan dari sebuah proses atau transaksi data ke dalam bentuk file yang dapat disimpan untuk keperluan tertentu. Report atau laporan yang dimaksud dalam tutorial ini adalah laporan PDF.

ada pemrograman PHP, secara fungsi kita hanya dapat melakukan manipulasi data pada proses CRUD (Create, Read, Update, dan Delete) dari dan ke database, masing-masing proses tersebut diimplementasikan melaui GUI pada sebuah web browser. Kemudian bagaimana sistem report / pelaporannya agar data tersebut dapat disimpan dalam bentuk file? Tentunya dengan cara export data dari database, kendalanya PHP tidak dapat melakukan itu sendiri. So, untuk dapat melakukan / membuat laporan pdf dengan PHP maka diperlukan sebuah library, yautu library FPDF. Library ini dapat Anda didownload melalui sumbernya. Untuk belajar CRUD seperti input data silahkan ikuti tutorial cara membuat form input data dengan PHP MySQL.

Tutorial cara membuat laporan PDF dengan PHP dan MySQL ini kami lakukan pada sistem operasi Windows 7 dan menggunakan web server XAMPP ver 3.2.1 include dengan database MySQL. Untuk mengikuti tutorial ini pastikan komputer anda telah terinstall web server XAMPP tersebut.
